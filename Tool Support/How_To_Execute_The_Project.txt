1. Install Eclipse Helios and Java 1.6 SDK tool.
2. Install maven from the http://download.eclipse.org/technology/m2e/releases/1.3 from the "Intall New Software" in Eclipse.
3. Install Ant 1.6.4 from the site add the path to ANT_HOME and ANT_OPTS(-Xmx1024m).
4. JAVA_HOME and PATH to include JDK path.
5. Checkout the repository and import the project
6. Build the project with Ant by doing a right click on the build.xml file and 
 modifying the VM arguments in Ant Build
-Xms512m
-Xmx1024m
